# php-todo-list


## Init Database SQL

```sql
CREATE TABLE `crud-example`.`tasks` (`id` INT UNSIGNED NOT NULL AUTO_INCREMENT , `text` VARCHAR(255) NOT NULL , `completed` TINYINT(1) NOT NULL DEFAULT '0' , `created_at` TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP , `updated_at` TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP , PRIMARY KEY (`id`)) ENGINE = InnoDB;
```
